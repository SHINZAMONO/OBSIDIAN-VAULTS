# 运算符

## 运算符 operator
Java 语言支持如下运算符：
* 算术运算符：+，-，* ，/，%，++，--
* 赋值运算符：=
* 关系运算符：<，>，>=，<=，== ，!=instanceof
* 逻辑运算符：&&，||，!
* 位运算符：&，|，^，~，>>，<<，>>>
* 条件运算符：?，:
* 扩展赋值运算符：+=，-=， * = ，/=
## 二元运算符
两个操作数
### 整数运算
如果两个操作数有一个为Long，则结果也为long
没有long时，结果为int。即使操作数全为short，byte，结果也是int
### 浮点运算
如果两个操作数有一个为double，则结果为double
如果两个操作数都是float，则结果为float
### 关系运算符
返回布尔值！
