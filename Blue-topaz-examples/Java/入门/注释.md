# 注释

## 单行注释
```Java
// 单行注释
```
## 多行注释
```Java
/* 多行注释 */
```
## 文档注释 Java Doc
```Java
/**
 *
 * @description: 
 * @author: ${USER}
 * @date: ${YEAR}-${MONTH}-${DAY} ${TIME}
 * @version: 1.0
 * @email: xincanzhe2333@hotmail.com
 */
```
⚠️警告：在Java 中，/**/ 注释不能成套。也就是说，不能简单地把代码用 /* 和 */ 括起 来作为注释，因为这段代码本身可能也包含一个* 界定符。
